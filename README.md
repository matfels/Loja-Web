Olá, este é um projeto de desenvolvimento WEB utilizando JavaScript, CSS e HTML. 

Objetivo: 
Apresentar o meu desenvolvimento na integração das tres linguegens. 

HTML: O arquivo "Loja.html" está a estrutura da página web, na pasta "img" estão os arquivos das imagens.
CSS: Na pasta CSS está o arquivo CSS que utilizei para definir a aparencia da página.
JavaScript: Na pasta JS esta o arquivo JavaScript que utilizei para tornar a aplicação dinâmina dos botões e calculos apresentados na página.
